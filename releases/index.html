Releases

A history of updates and new games on DailyJamm.

v1.9.0 April 15, 2026

Stats have been reset across all games. The stats system was rebuilt to track guess distributions and richer lifetime data. Because previous stats were recorded without this structure, all game stats have been cleared so every player starts on equal footing with accurate numbers from this point forward. Daily game progress (today's puzzle, chip balance) was not affected.

  • Stats keys bumped to _v2 - previous counts, streaks, and chip history cleared site-wide
  • Guess distribution charts now appear in the Stats modal for Themedle and Spelldle, showing how often you win on each attempt
  • Share Stats button added to every game's Stats modal - copies a shareable text summary to clipboard
  • Game-over result panels redesigned: compact panel with two side-by-side buttons - Share Results and See Stats - replacing the old inline streak/played mini-display
  • Themedle: fixed a crash on page load when previously stored stats data was malformed, which silently prevented the result panel from rendering
  • Themedle: removed the redundant countdown from the page header; the timer now only appears in the result panel after the game ends
  • Themedle: result panel now uses the same compact card style as all other games for visual consistency
▶

v1.8.0 April 3, 2026

  • Stats button (bar chart icon) added to all game pages - shows lifetime stats modal
  • Standardized header layout: Stats and ? (How to Play) buttons always appear top-right in the fixed header on every game page
  • BlackJackdle: improved active hand glow during split
  • Roulettedle spin history modal: last 10 spins, red/black and odd/even split bars, spins since last 0/00, and all-time net
